
Imports multiple WIF Private Keys into the wallet to be used to sign
transactions by an account.

example: import_keys

dQ9qMcEAC43kujbfjueTHpVapX1Kzq2n"]

Parameters:
    wif_keys: the WIF Private Keys array to import (type: const
	vector<string> &)
